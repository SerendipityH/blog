redis的持久化

发表于 2019-03-10 | 分类于 redis
字数统计: 166

Redis的所有数据都是保存到内存中的。

Rdb：快照形式，定期把内存中当前时刻的数据保存到磁盘。Redis默认支持的持久化方案。

可根据情况自己配置

aof形式：append only file。

把所有对redis数据库操作的命令，增删改操作的命令。保存到文件中。数据库恢复时把所有的命令执行一遍即可。
aof形式的完整性更高一些，appendfsync everysec 为每秒存一次，但会影响redis的性能。

appendonly no改为yes开启aof形式

两种持久化方案同时开启使用aof文件来恢复数据库。
